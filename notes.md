## Golang Notes

## Variables

- Variables are declared using the `var` keyword
- Variables can also be declared using the `:=` operator
- Multiple variables can be declared in a single line using the `var` keyword

```go
func main() {
  var name string = "John Doe"
  var age int = 30
  var isCool = true
  isCool = false
  // Shorthand
  name := "John Doe"
  size := 1.3
  name, email := "John Doe", "
}
```
